Lesson 02: सन्धि:

Part 1: Grammar

सन्धयः त्रयः सन्ति | Sandhi is of three types:


  • स्वरसन्धिः
    द्वयोः स्वरयोः मेलनेन यः विकारः भवति सः स्वरसन्धिः। The change that occurs by the combination of two vowels is called vowel sandhi.
  • व्यञ्जनसन्धिः
    व्यञ्जनेन सह स्वरस्य व्यञ्जनस्य वा मेलनेन यः विकारः भवति सः व्यञ्जनसन्धिः। The change that occurs when a consonant combines with either a vowel or another consonant is called consonant sandhi.
  • विसर्गसन्धिः
    विसर्गेण सह स्वरस्य व्यञ्जनस्य वा मेलनेन यः विकारः भवति सः विसर्गसन्धिः। The change that occurs when a visarga combines with either a vowel or a consonant is called visarga sandhi.

दीर्घसन्धिः | नियमः - अकः सवर्णे दीर्घः ६.१.१०१

When अक् (अ, आ, इ, ई, उ, ऊ, ऋ, ॠ, ऌ) is followed by the same vowel, they merge into the corresponding दीर्घ vowel.

Patterns: अ/आ + अ/आ → आ, इ/ई + इ/ई → ई, उ/ऊ + उ/ऊ → ऊ, ऋ/ॠ + ऋ/ॠ → ॠ

Combination Example
अ/आ + अ/आ → आ मुर + अरिः = मुरारिः, हिम + आलय: = हिमालय:, विद्या + अर्थी = विद्यार्थी, दया + आनन्दः = दयानन्दः
इ/ई + इ/ई → ई कपि + इन्द्रः = कपीन्द्रः
उ/ऊ + उ/ऊ → ऊ गुरु + उपदेशः = गुरूपदेशः
ऋ/ॠ + ऋ/ॠ → ॠ पितृ + ऋणम् = पितॄणम्

दीर्घसन्धिः | नियमः - अकः सवर्णे दीर्घः ६.१.१०१

When अक् (अ, आ, इ, ई, उ, ऊ, ऋ, ॠ, ऌ) is followed by the same ह्रस्व or दीर्घ, they merge into their corresponding दीर्घ form.

Patterns: अ/आ + अ/आ → आ, इ/ई + इ/ई → ई, उ/ऊ + उ/ऊ → ऊ, ऋ + ऋ/ऌ → ॠ

e.g.: मुर + अरिः = मुरारिः, हिम + आलय: = हिमालय:, विद्या + अर्थी = विद्यार्थी, दया + आनन्दः = दयानन्दः, कपि + इन्द्रः = कपीन्द्रः, गुरु + उपदेशः = गुरूपदेशः, पितृ + ऋणम् = पितॄणम्


गुण-सन्धिः | नियमः - आदुगुणः ६.१.८७

When अ or आ is followed by इ/ई, उ/ऊ, or ऋ/ॠ, they merge into ए, ओ and अर् respectively.

Patterns: अ/आ + इ/ई → ए, अ/आ + उ/ऊ → ओ, अ/आ + ऋ/ॠ → अर्

e.g.: उप + इन्द्रः = उपेन्द्रः, सूर्य + उदयः = सूर्योदयः, महा + ऋषिः = महर्षिः


वृद्धि-सन्धिः | नियमः - वृद्धिरेचि ६.१.८८

When अ or आ is followed by ए/ऐ or ओ/औ, they merge into their corresponding वृद्धि form.

Patterns: अ/आ + ए/ऐ → ऐ, अ/आ + ओ/औ → औ

e.g.: तत्र + एव = तत्रैव, जल + ओघः = जलौघः


अयादि-सन्धिः | नियमः - एचोऽयवायावः ६.१.७८

When ए, ऐ, ओ or औ are followed by any vowel, they transform into अय्, आय्, अव् and आव् respectively.

Patterns: ए + अ → अय, ऐ + अ → आय, ओ + अ → अव, औ + अ → आव

e.g.: ने + अनम् = नयनम्, पो + अनः = पवनः


पूर्वरूप सन्धि: | नियमः - एङः पदान्तादति ६.१.१०९

When a word ending with ए/ओ is followed by अ, the अ is replaced by अवग्रह.

Patterns: ए/ओ + अ → एऽ / ओऽ

e.g.: जठरे + अग्निः = जठरेऽग्निः


यण्-सन्धिः | नियमः - इको यणचि ६.१.७७

When इक् (इ, उ, ऋ, ऌ) is followed by a different vowel, it transforms into यण् (य्, व्, र्, ल्).

Patterns: इ → य्, उ → व्, ऋ → र्, ऌ → ल्

e.g.: इति + आदि = इत्यादि, मूर्ध्नि + आदित्याः = मूर्ध्न्यादित्याः


विसर्ग-रेफ-सन्धिः | नियमः - इचोऽशि विसर्गस्य रेफः ८.३.१४

When a विसर्गः preceded by non-अ/आ vowel is followed by a voiced sound, it becomes र्.

Patterns: ः → र्

e.g.: पादयोः + विष्णुः = पादयोर्विष्णुः


विसर्ग-उत्व-सन्धिः | नियमः - अतो रोरप्लुतादप्लुते ६.१.११३

When विसर्ग is preceded by अ and followed by a voiced sound, it becomes ओ.

Patterns: अः + (voiced) → ओ, अः + अ → ओऽ

e.g.: सर्वतः + वायुः = सर्वतो वायुः, सर्वतः + अग्निः = सर्वतोऽग्निः


विसर्ग-सत्त्व-सन्धिः | नियमः - विसर्जनीयस्य सः ८.३.३४

When विसर्गः is followed by hard consonants, it becomes सibilant.

Patterns: ः + त् → स्, ः + च् → श्, ः + ट् → ष्

e.g.: नमः + ते = नमस्ते, रामः + च = रामश्च


विसर्ग-प्रकृतिभावः | नियमः - वा शरि ८.३.३६

Visarga may remain unchanged before sibilants.

Patterns: ः + श्/ष्/स् → optional

e.g.: पुरतः + शूली = पुरतः शूली


जश्त्व-सन्धिः ८.२.३९

Final consonants change to soft वर्ग equivalents before voiced sounds.

Patterns: त् → द्, क् → ग्, प् → ब्

e.g.: तत् + भवति = तद्भवति


अनुस्वार-सन्धिः ८.३.२३

म्/न् becomes anusvāra or corresponding nasal before consonants.

Patterns: म् → ं, न् → ं

e.g.: सम् + करोतिः = सङ्करोति
